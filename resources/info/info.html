Icon Reference Guide

A quick reference for all icons and symbols used throughout the app.
Toolbar
Add Link — Add a new HTTP download by URL
Torrent — Import a .torrent file or magnet link
Options — Open app settings
Info — This reference guide
Exit — Close the application
Speed Column — Engine Indicator
Single bolt (green) — HTTP single-thread download
Double bolt (gold) — HTTP multi-thread download (16 threads)
Torrent icon (amber) — BitTorrent P2P download
Download Actions
Pause — Pause an active download
Resume — Resume a paused download
Cancel — Stop and cancel a download
Open Folder — Open the download folder
Open File — Open the downloaded file
Delete — Remove the download record from history
Status Colors
Yellow — Downloading / In progress
Green — Completed successfully
Red — Error / Cancelled / Failed
Gray — Idle / Not applicable
Table Sorting
▲ Up arrow — Sorted ascending (A→Z, 0→100, oldest first)
▼ Down arrow — Sorted descending (Z→A, 100→0, newest first)
Resource Monitor (Top Right)
CPU — App CPU usage percentage
RAM — App memory usage in MB
↓ Green arrow — Total download speed
↑ Yellow arrow — Total upload speed (torrent only)
Peers Column
15 Yellow number — Active peers connected (torrent downloads)
- Dash (gray) — Not applicable (HTTP downloads)
