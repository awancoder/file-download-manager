# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by emailing us at [awan@awancoder.com] or by opening an issue with the label `security`.

We will respond as quickly as possible to address the issue.

## Supported Versions

| Version | Supported          |
| ------- | ----------------- |
| latest  | :white_check_mark:|

## Security Updates

We will announce security updates in the repository's release notes. Please ensure you are using the latest version.

## Responsible Disclosure

We request that you give us the opportunity to address the issue before disclosing it publicly.
